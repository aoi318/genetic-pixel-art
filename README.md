### 追加予定ロードマップ  

#### 1. 画像アップロード対応
- ユーザーがターゲット画像をアップロード
- 選択グリッドサイズに自動リサイズ
- デフォルト画像も保持
- トグル: 「カスタム画像使用: ON/OFF」

#### 2. 高度なフィットネス関数
- MSEに加えてエッジ検出、色ヒストグラムなどを選択可能
- トグル: 「高度フィットネス: ON/OFF」（OFF時はシンプルMSE）

#### 3. インタラクティブ進化モード
- 各世代で複数候補を表示し、ユーザーが好きな個体を選択
- トグル: 「インタラクティブモード: ON/OFF」

#### 4. 100世代ごとの推移をGIF出力
- 進化過程を100世代ごとにスナップショット
- 終了後にGIFとしてダウンロード可能
- トグル: 「GIF出力: ON/OFF」

#### 5. リアルタイム進化グラフ
- フィットネス推移をグラフ表示（recharts使用）
- 世代ごとの最良/平均/最悪fitnessをライン表示
- 多様性指標（個体間の差異）も表示可能
- トグル: 「グラフ表示: ON/OFF」

#### 6. 差分ヒートマップ表示
- ターゲット画像と進化画像の差分を可視化
- 誤差が大きい部分を赤く、小さい部分を青く表示
- リアルタイムで更新され、改善箇所が一目瞭然
- トグル: 「ヒートマップ表示: ON/OFF」

#### 7. ベスト画像ダウンロード機能
- 現在のベスト個体をPNG形式で保存
- ファイル名に世代数とfitness値を自動付与
- オリジナルサイズまたは拡大版を選択可能
- ボタン: 「画像をダウンロード」

#### 8. ステップ実行モード
- 1世代ずつ手動で進めるモード
- 各ステップで選択・交叉・突然変異の過程を可視化
- デバッグや教育目的に最適
- トグル: 「ステップモード: ON/OFF」（ON時は「次の世代」ボタン表示）
