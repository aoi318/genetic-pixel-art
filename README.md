### 追加予定ロードマップ  

#### 1. パラメーター変更機能 + 自動変異率調整
- UIで人口サイズ（例: 100〜1000）と変異率（例: 0.01〜0.1）を調整可能
- 「成績が良くなったら変異率を減らす」自動調整モード追加
- トグル: 「自動変異率調整: ON/OFF」

#### 2. グリッドサイズ選択機能
- スタート前に32x32 / 64x64 / 128x128を選択可能
- 高解像度時は計算量警告を表示
- トグル: ドロップダウンでサイズ選択（デフォルト32x32）

#### 3. 画像アップロード対応
- ユーザーがターゲット画像をアップロード
- 選択グリッドサイズに自動リサイズ
- デフォルト画像も保持
- トグル: 「カスタム画像使用: ON/OFF」

#### 4. 並列処理対応 + 速度比較機能
- Web Workers + WASM Threadsでフィットネス計算を並列化
- オンオフで並列/シングル切り替え
- 実行後に処理時間比較を表示
- トグル: 「並列処理: ON/OFF」（ON時に速度比較自動表示）

#### 5. 人口サイズ・世代数の大幅増加 + 高速進化モード
- 人口5000以上、世代10万以上対応
- 「高速モード」で表示更新を抑えて最大速度優先
- トグル: 「高速進化モード: ON/OFF」

#### 6. 高度なフィットネス関数
- MSEに加えてエッジ検出、色ヒストグラムなどを選択可能
- トグル: 「高度フィットネス: ON/OFF」（OFF時はシンプルMSE）

#### 7. 制限付き色パレットモード
- 8色/16色などレトロパレット限定モード
- パレットを自動抽出または手動選択
- トグル: 「パレット制限: ON/OFF」（ON時は色数選択）

#### 8. インタラクティブ進化モード
- 各世代で複数候補を表示し、ユーザーが好きな個体を選択
- トグル: 「インタラクティブモード: ON/OFF」

#### 9. 100世代ごとの推移をGIF出力
- 進化過程を100世代ごとにスナップショット
- 終了後にGIFとしてダウンロード可能
- トグル: 「GIF出力: ON/OFF」
